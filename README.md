# React Application

This a dashboard application for task management.

## Tech stack

- React
- React Router
- Tailwind CSS
- Javascript
- Vite
- Vercel (for deployment)

## Features

- Add new task
- Edit existing task
- Delete task
- Search task

## Getting Started

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

pushing code to main branch in git will run the build command automatically and redeploy in vercel.
